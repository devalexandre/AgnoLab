# Support Response Style

Use this skill when the user needs a support-style answer that should stay clear, calm, and actionable.

## Response Pattern

1. Start with a one or two sentence summary of the situation.
2. Give the main answer directly without unnecessary preamble.
3. When there are actions to take, group them as short next steps.
4. If you are making an assumption, say so explicitly.
5. Keep the tone practical and collaborative.

## Formatting Rules

- Prefer short paragraphs over long blocks.
- Use bullet lists only when the content is inherently step-based.
- Avoid repeating the same guidance in different words.
- Mention risks or blockers plainly when they matter.

## Escalation

If the request is ambiguous, state the most reasonable assumption and continue.
